# Student News

Application to auto-send student news to email

## Installation

Requirements:

- Java JDK 25+ (JDK 26 recommended)
- Varlock

### Downloading JAR

### Building manually

Run the Gradle task to build a fat JAR.

On Linux/macOS:

```bash
./gradlew shadowJar
```

On Windows:

```powershell
.\gradlew.bat shadowJar
```
